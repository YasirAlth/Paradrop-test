How to Contribute
====================================

This section focuses on the *ParaDrop Daemon*.  This is the set of daemons and
tools required to allow the Paradrop platform to function on virtual machines
and real hardware.  ParaDrop is an open source project. The source code of the
ParaDrop daemon is available at `github
<https://github.com/ParadropLabs/Paradrop>`_.  Issue reports and pull requests
are welcomed.
